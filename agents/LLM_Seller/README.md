# Project

This project involves quarter sales.

# Prerequisites

This project requires Python 3.10 or later. All depedndent packages and their versions are metioned in requirement.txt.

# Setup Instructions

Go to root folder \QtrAnalysis and run the below command

pip install -r requirements.txt


Once requirments are installed run main file

python main.py

# Output

All execution instruction will be shown as per process. All output graphs and matrics will be published in output folder


# Folder Structure

data/ — Contains input data files like qtr_revenue.xlsx.

output/ — Stores results like plots and CSVs.

main.py — The main script to execute.

requirements.txt — Lists required Python packages.

README.md — Setup 
and usage instructions.
